This guide provides tutorials for installing the software necessary for the boot camp. All installation should be completed prior to the beginning of the boot camp. Note that installation must be done as a root user or a user with certain administrator privileges. If you run into problems with installation, contact your boot camp's host as soon as possible.

All installation techniques have been tested using Mac OSX 10.7.4.
Mac OSX 10.7.4 already comes with the following installed: bash shell, vi and nano text editors, Python 2.7, Python setuptools, SQLite3.

The Shell

Mac OSX 10.7.4 comes with bash shell (the terminal) installed. You can access it several ways:

(1) In the Applications/Utils folder on your Mac, you can find Terminal. Double click on it to open the shell.


(2) Using the Search feature on your Mac (upper right corner), search for "terminal:


Open the terminal. You should see a new terminal screen pop up on your computer. For ease of use, you may want to pin the terminal to your sidebar permanently.


Install make

If you do not already have access to make from within your shell, you will need to install XCode (which is free)

  • Go to the Apple app store
  • Search for XCode
  • Click Free
  • Click Install App

Once XCode has installed

  • Click Applications
  • Click XCode
  • Select XCode => Preferences...
  • Click Downloads
  • Select Command Line Tools
  • Click Install

You will now be able to run make within your bash shell.

Install Python nose

Run

$ easy_install nose

And check

$ nosetests

---------------------------------------------------------------------- Ran 0 tests in 0.003s

OK
